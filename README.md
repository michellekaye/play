## Play

Play is a side project to showcase my collection of board games.

Visit [play.michelle-kaye.com](http://play.michelle-kaye.com)

## Getting Started

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies

- Next.js
- React
- Typescript
- SCSS
- Github actions
- AWS S3
